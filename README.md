#Crypto

Crypto - CLI утилита для шифрования данных. Поддерживается работа с 3 шифрами:

- **Цезаря**
- **Вернама**
- **Виженера**

##Пример

```js
 node index.js --input "Hello, World" --cipher "vigenere"  --key "abc" --encrypt

```

## Установка
1. Клонировать репозиторий
2. Выполнить команду `yarn install`

## Требования

1. Node.js 7.1.0
